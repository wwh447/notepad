F1打开小编的帮助主页
亲可以随时按该热键打开小编的帮助主页，这是小编的意念存储，智慧结晶。
F2设置书签标记
这个功能十分有用，特别是当你修改一个代码量较大的文件时格外好使。按下这个热键，光标所在行会被添加一个蓝色小球，这就是书签。你可以从文档的任何位置，快速回到书签标注的行。只要不单独关闭设置书签的这个文档，即使关闭主程序，下次打开书签依然在那里。
Ctrl + F2跳转到下一个书签
可以快速跳到下一个设有书签标识的行。
Shift + F2跳转到上一个书签
可以快速跳到上一个设有书签标识的行。
Ctrl + W关闭正在编辑的文档
该热键可以快速关闭当前正在编辑的文档，未保存将弹出提示，已保存将直接被关闭。和直接关闭编辑器不同，直接关闭编辑器当前所有打开的文档，下次还会被自动打开(工作记忆功能)。但通过这种方式关闭的文档就真正被释放了，可以在[文件]菜单里看到近期打开过的文档，并且点选重新打开。
Ctrl + Tab切换正在编辑的文档
当打开多个文档的时候，可以通过该热键快速在打开文件里切换。长按该热键不放，会显示所有已打开的文档，这时不要放开 Ctrl 键，松开再多次轻巧Tab键可慢慢挑选文档。
Tab代码缩进
单行时光标停在句首，按该热键可缩进代码；多行时需选中要缩进的片段，按热键可缩进代码。需要注意的是，多行缩进时并不是必须要包含所有字符，选中多行即可。
Shift + Tab代码退格
和缩进相反的操作，按该热键可将代码片段退格。操作方法和代码缩进相同，不再赘述。
Ctrl + D复制当前行
快速把当前行内容，复制并黏贴到下一行，简化常用操作。
Ctrl + F搜索内容
可以用来查找指定代码片段，支持正则表达式，是最实用的热键之一。
Ctrl + H替换内容
可以将匹配的代码片段替换为指定的代码，支持正则、选取范围。
Ctrl + Shift + F在文件中查找片段
用途和搜索内容功能类似，但功能更强大，可以扫描指定目录内的所有文件，并且可以过滤文件扩展名，同时支持将所有匹配到的内容替换。
F3查找下一个
需要先选中一段文本，接着可以快速查找到下一个与选中片段同的代码。
Shift + F3查找上一个
需要先选中一段文本，接着可以快速查找到上一个与选中片段同的代码。
Ctrl + G行定位
将光标快速定位到文档指定的行号，是最实用的热键之一。
